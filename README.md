# next-validator

Forked [GitHub - yiminghe/async-validator: validate form asynchronous](https://github.com/yiminghe/async-validator)

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
